🚀 Configurar Supabase en 5 pasos
1
Crear cuenta gratuita en Supabase

Ve a supabase.com → Start your project → Inicia con GitHub o correo.
El plan gratuito (Free) soporta 500 MB de datos y hasta 50.000 filas. Más que suficiente para un semestre.

2
Crear un proyecto nuevo

Clic en "New project". Configura:

  • Name: tga05-neurobiz
  • Database password: cualquiera segura (la necesitarás solo para acceso directo)
  • Region: South America (São Paulo) — más cerca de Colombia

Espera ~2 minutos mientras el proyecto se aprovisiona.

3
Ejecutar el esquema SQL

En el panel de tu proyecto: menú lateral SQL Editor → "New query".
Abre el archivo setup/supabase_schema.sql de esta carpeta, copia todo el contenido y pégalo en el editor.
Clic en "Run" (o Ctrl+Enter).

✅

Deberías ver: "Tabla student_progress creada correctamente ✅"

4
Obtener la URL y la clave anónima

Menú lateral → Project Settings → API.

Copia estos dos valores:

CampoDónde estáEjemplo
Project URL Sección "Project URL" https://abcxyz.supabase.co
anon / public key Sección "Project API keys" eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...
⚠️

La clave anon/public es segura para incluir en HTML. Nunca uses la clave service_role en el frontend.

5
Configurar la URL y clave en los materiales

Pega los valores aquí abajo y haz clic en "Guardar configuración". Se almacenarán en localStorage de este navegador y activarán la sincronización en todos los materiales del módulo que estén en este mismo navegador.

Opción A — Vista rápida en Supabase

1

En tu proyecto Supabase → Table Editor → tabla student_progress.

2

O usa el SQL Editor con esta consulta de resumen:

SELECT * FROM resumen_docente ORDER BY xp_total DESC;

Verás: estudiante, semanas visitadas, XP total, % de avance, última actividad.

Opción B — Consultas útiles

-- Estudiantes con menos del 30% de avance
SELECT student_name, pct_avance, xp_total
FROM resumen_docente
WHERE pct_avance < 30
ORDER BY pct_avance;

-- Quiz scores de la semana 3
SELECT student_name, quiz_score, xp
FROM student_progress
WHERE semana = 3
ORDER BY quiz_score DESC;

-- Actividad por semana
SELECT semana,
       COUNT(*) AS estudiantes_activos,
       AVG(xp) AS xp_promedio
FROM student_progress
GROUP BY semana
ORDER BY semana;
