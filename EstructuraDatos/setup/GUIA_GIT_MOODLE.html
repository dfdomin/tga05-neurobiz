🔀 Publicar en GitHub Pages por primera vez

Ejecuta estos comandos una sola vez desde la carpeta EstructuraDatos/.

Paso 1 — Preparar el repositorio local

# Ir a la carpeta del proyecto
cd "/Users/diegodomingueztapia/Library/CloudStorage/OneDrive-Unibarranquilla/DiegoIcloud/2026/copilot/EstructuraDatos"

# Inicializar git (solo la primera vez)
git init

# Agregar todos los archivos
git add .

# Primer commit
git commit -m "TGA05 NeuroBiz - Material completo con identidad IUB"

Paso 2 — Crear repositorio en GitHub y conectar

1
Ve a github.com/new y crea un repositorio llamado tga05-neurobiz (sin README, sin .gitignore)
2
Copia la URL del repositorio: https://github.com/TU_USUARIO/tga05-neurobiz.git
# Conectar con GitHub (reemplaza TU_USUARIO)
git remote add origin https://github.com/TU_USUARIO/tga05-neurobiz.git

# Subir al repositorio
git push -u origin main

Paso 3 — Activar GitHub Pages

1
En GitHub: Settings → Pages
2
Source: Deploy from a branch → Branch: main → Folder: / (root) → Save
3
En ~2 minutos tu sitio estará en: https://TU_USUARIO.github.io/tga05-neurobiz/
⚠️
Antes de subir: edita js/supabase-config.js con tu URL y clave anon de Supabase (Project Settings → API en supabase.com). La clave anon es pública y segura con RLS activado.

🔄 Actualizar el sitio (próximas veces)

Cada vez que modifiques archivos, usa solo esto:

cd "/Users/diegodomingueztapia/Library/CloudStorage/OneDrive-Unibarranquilla/DiegoIcloud/2026/copilot/EstructuraDatos"

📌 Cómo agregar cada semana como URL en Moodle

1
En tu curso Moodle → Activar edición (botón arriba a la derecha)
2
En la pestaña/semana correspondiente → "Añadir actividad o recurso"
3
Selecciona URL en la lista de recursos
4
Configura así:
• Nombre: 📘 Semana 1 – Introducción a Estructuras de Datos
• URL externa: https://TU_USUARIO.github.io/tga05-neurobiz/semana1/
• Apariencia → Mostrar: En nueva ventana (evita problemas con iframe de Moodle)
5
Guardar y repetir para las 14 semanas
💡
Tip: Usa el mismo ícono en el nombre para diferenciar tipo de contenido: 📘 Material de estudio · 📝 Notas del profesor · 📊 Dashboard.
