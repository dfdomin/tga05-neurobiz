**Semana:** 11  
**Modalidad:** HP · Bloque 2 de 2 (mismo día; descanso 10 min entre bloques)

**Tema:** Laboratorio de trace tables + primer programa Python en Thonny con datos NeuroBiz

## 1. Propósito observable
Al finalizar el bloque, el estudiante será capaz de:
- Completar 3 trace tables de algoritmos selectivos y repetitivos.
- Explicar verbalmente por qué cambió una variable entre un paso y otro.
- Crear, guardar y ejecutar un archivo Python con datos reales de NeuroBiz.
- Usar `print()`, variables numéricas y cadenas con formato `f-string`.
- Entregar evidencia dual: trace table resuelto + archivo `.py` funcional.

## 7. Nota neurodivergencia
La combinación **trace table + primer código corto** favorece a estudiantes que necesitan ver el paso intermedio entre idea y programa. En presencia de discalculia, escribir el valor de cada variable evita sobrecarga y mejora la autoverificación. Se debe decir explícitamente: **“si lo puedes rastrear en tabla, luego lo puedes programar con menos ansiedad.”**

## 8. Contingencias
- **Si faltan equipos:** trabajar en parejas fijas y exigir que ambos escriban al menos una parte del código.
- **Si un estudiante no logró abrir Thonny:** usar equipo espejo con compañero y entregar evidencia compartida del bloque.
- **Si el grupo termina antes:** agregar una variable `uso_uib = 94.7` e imprimir una segunda línea con el porcentaje de uso.
- **Si el grupo se atrasa:** priorizar ejercicios Python 1 y 2; el `input()` queda como reto opcional.
- **Si se pierde energía o internet:** completar los ejercicios en cuaderno con sintaxis Python copiada del tablero.
