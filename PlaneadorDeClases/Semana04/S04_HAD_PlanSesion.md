## 6. Contenido exacto que debe quedar visible en tablero o diapositiva

### 6.1 Definiciones cortas
- **Suma acumulada:** agregar valores uno tras otro en la misma variable.
- **Promedio:** suma total dividida entre cantidad de datos.
- **Máximo:** valor más alto del conjunto.
- **Mínimo:** valor más bajo del conjunto.
- **Conteo:** número de casos que cumplen una condición o que fueron registrados.

### 6.2 Notación matemática
- `Promedio = Σ valores / n`
- `Máximo = mayor valor del conjunto`
- `Mínimo = menor valor del conjunto`
- `Conteo = número de elementos`

### 6.3 Notación de pseudocódigo
```pseint
suma <- suma + valor
promedio <- suma / cantidad
contador <- contador + 1
Si valor > maximo Entonces
    maximo <- valor
FinSi
Si valor < minimo Entonces
    minimo <- valor
FinSi
```

### 6.4 Cálculo guía que no debe borrarse
```text
Promedio N1 = (85.0 + 98.6 + 75.0 + 96.3 + 96.7 + 59.7 + 95.8 + 100.0) / 8
            = 707.1 / 8
            = 88.4%
```

## 7. Tareas de los grupos (bloque 5)
- **Grupo 1:** calcular promedio de N4 e identificar terapeutas en riesgo por **N4 > 15%**.
- **Grupo 2:** calcular promedio de satisfacción e identificar quiénes quedaron por debajo del promedio.
- **Grupo 3:** perspectiva UIB — calcular **U1** y **U2**.
- **Grupo 4:** perspectiva UIB — calcular **U3** y discutir qué implica que sea **16.7%**.

**Instrucción docente fija:** “Cada grupo entrega 1 cifra, 1 interpretación y 1 recomendación. Nada más.”

## 8. Banco sugerido de 8 preguntas para Quizizz Playground
1. La operación `suma <- suma + valor` corresponde a: acumulación / promedio / salida / conteo.  
2. El operador estadístico que busca el valor más alto es: máximo / mínimo / conteo / lógica.  
3. El promedio de N1 de NeuroBiz es: 88.4% / 75.0% / 94.7% / 16.7%.  
4. El terapeuta con **N1 máximo** es: T08 / T06 / T03 / T01.  
5. El terapeuta con **N1 mínimo** es: T06 / T08 / T05 / T02.  
6. El promedio de N4 del caso es: 9.7% / 16.7% / 88.4% / 60.0%.  
7. U3 mide: mejora académica / uso de contrato / costo por estudiante / satisfacción.  
8. `sum(), max(), min(), len()` pertenecen al ejemplo de: Python / Excel / Word / draw.io.  

**Clave:** 1 acumulación, 2 máximo, 3 88.4%, 4 T08, 5 T06, 6 9.7%, 7 mejora académica, 8 Python.

## 9. Contingencias permitidas (únicas rutas de desvío)
> **Protocolo ACCC docente:** si algo falla, elegir una sola ruta y anunciarla con la frase: **“Activamos plan alterno; mantenemos objetivo, cambia solo la herramienta.”**

- **Si Quizizz no conecta:** reemplazar por 8 preguntas orales de mano alzada y asignar puntos de ClassDojo en tiempo real.
- **Si PSeInt se cae:** abrir editor de texto y escribir pseudocódigo en `.txt`; la lógica no cambia.
- **Si la actividad grupal se alarga:** omitir la presentación de 1 grupo y pedirle publicar su respuesta en aula virtual.
- **Si falla el proyector:** mover todo a tablero; el código de Quizizz se dicta y el pseudocódigo se escribe a mano.
- **Si el grupo está poco comprometido:** convertir las preguntas en reto entre mesas de 60 segundos y otorgar **Participación** + **Curiosidad**.
- **Si van 8–10 minutos tarde:** conservar bloques 2, 4 y 7; reducir Quizizz a 5 preguntas y Python a 3 minutos.
